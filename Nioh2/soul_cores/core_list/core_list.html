Description

Below is a list of all of the Soul Cores in Nioh 2. It is split into Base Game Soul Cores and DLC Soul Cores. Each Soul Core is 1 of 3 Yokai Forms. Soul Cores that are colored RED are of the BRUTE Yokai Form. Soul Cores that are colored PURPLE are of the PHANTOM Yokai Form. Soul Cores that are colored BLUE are of the FERAL Yokai Form.

Total number of Soul Cores: 78 Soul Cores

Number of Soul Cores in the Base Game: 51
Number of Soul Cores in The Tengu's Desciple (DLC 1): 6
Number of Soul Cores in The Darkness in the Capital (DLC 2): 9
Number of Soul Cores in The First Samurai (DLC 3): 12
