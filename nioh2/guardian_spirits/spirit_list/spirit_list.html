Guardian Spirit Combination List

Combining different spirits results in an extra special effect, this effect is based on the spirit pairing. All Gurdian Spirits are broken up into Element type + Yokai type, EX) Makami is a Fire type + Brute type. There are 5 Elements: Fire, Water, Lightning, Purity, and Corruption. There are 3 Yokai Forms: Brute, Phantom, and Feral. A general formula is followed to get the special effect. Below is a table containing the combinations of Guardian Spirits and the special effect.

General Combinations
Guardian Spirit Combination Special Effect
Both Guaridan Spirits are the same Elemental Type (Excludes Purity and Corruption) Increase Element Damage +5.0 %
Purity Element + Corruption Element Anima Bonus (Confused Enemy) D
Both Guaridan Spirits are the same Yokai Type Yokai Ability Damage (Type) +5.0 %
Brute + Feral Ki Consumption (Yokai Shift) -6.0 %
Brute + Phantom Anima Charge (Strong Attack) +5.0 %
Phantom + Feral Anima Charge (Active Skill) +3.0 %
