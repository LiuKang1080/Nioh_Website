Basic Game Settings

Changing the Item Rarity Color Pattern

To change the color of rarity of item drops from the main menu: SYSTEM → BASIC GAME SETTINGS → 4th PAGE → RARITY COLOR PATTERN

Blue: Rare
Purple: Exotic
Green: Divine
Orange: Ethereal

To display the damage numbers that we recieve and deal to enemies, we set both options to ON: in SYSTEM → BASIC GAME SETTINGS → 1st PAGE

Online Game Settings

TODO: [ Complete Image of the Online Settings ]
General Online Settings Suggestions:

MATCHING AREA: Suggested Setting = UNLIMITED. By setting the Matching Area to Unlimited, there are higher chances to be matched with more people in Random Online Co-Op play. If there are connection issues / lag, it might improve when the setting is set on "Limited", but you will be matched with less people. Wait times might increase.

VISITOR'S ONLINE ID / HEALTH GAUGE: Setting suggestions: For ONLINE ID it is recommended to turn the setting to ON. For the HEALTH GAUGE it is recommended to set the setting to "Always Display". The visitor's health bar will always be displayed.

SECRET WORD: The Secret Word can be used to directly match with a user who has the same Secret Word. If you are directly matching with your friends and are inviting them for co-op through the Torii Gate via the Friends filtering option it is suggested to disable the Secret Word, especially if your co-op friend does not have a Secret Word / doesn't have the same Secret Word.

Number of Shortcut Sets

We can change the number of Shortcut Wheels by going to INDIVIDUAL GAME SETTINGS → 2nd PAGE


Hold to Switch Back to Set 1

This option might be helpful for some people. We can hold the button to switch sets to always go back to Set 1, regardless of what current Set is selected.
To enable this, we can go to INDIVIDUAL GAME SETTINGS → 2nd PAGE and set the option "Hold 'Switch Item Shortcut' to switch to set 1" to ON.
